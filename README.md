# 👗 Dressify GST

**Dressify GST** is a Java Swing-based desktop application designed to simulate a real-world dress shop billing and management system. It provides an intuitive, interactive graphical user interface (GUI) that streamlines the process of choosing apparel, managing a virtual shopping cart, and generating itemized tax invoices.

---

## ✨ Features

*   **Category-Based Browsing:** Select apparel seamlessly from multiple dedicated categories: **Male**, **Female**, and **Kids**.
*   **Dynamic Cart Management:** Add items to a virtual shopping cart, modify selections, and view real-time calculations.
*   **Dynamic Inventory Updates:** Ability to add new dress entries to the system on the fly.
*   **Automated GST Billing:** Automatic calculation of Goods and Services Tax (GST) applied to the final bill.
*   **Invoice Generation:** Formats and displays a clean, readable tax invoice for transaction records.
*   **Vibrant UI Design:** Features a modern look incorporating color-themed panels, images, and emojis to elevate the user experience.

---

## 🛠️ Tech Stack & Concepts

*   **Language:** Java (Core Java)
*   **GUI Framework:** Java Swing & AWT
*   **Architecture:** Object-Oriented Programming (OOP) principles (Inheritance, Encapsulation, Polymorphism)
*   **Data Management:** Efficient in-memory data structures for persistence, accuracy, and quick retrieval of dress details and transactions.

---

## 🚀 Getting Started

### Prerequisites
*   **Java Development Kit (JDK):** Version 8 or higher.
*   An IDE like **IntelliJ IDEA**, **Eclipse**, or **NetBeans** (optional, but recommended).

### Running the Application
1. **Clone the repository:**
   ```bash
   git clone [https://github.com/VigneshkumarS42156/Dress-shop-management.git](https://github.com/VigneshkumarS42156/Dress-shop-management.git)

```

2. **Navigate to the project directory:**
```bash
cd Dress-shop-management

```


3. **Compile the Java files:**
```bash
javac *.java

```


4. **Run the application:**
```bash
java Main 

```


*(Note: Replace `Main` with the actual name of your entry point class if it differs.)*
