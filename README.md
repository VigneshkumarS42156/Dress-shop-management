The Dressify GST project is a Java Swing-based desktop application developed to simulate a real-world dress shop billing and management system. It provides an intuitive and interactive user interface where users can select dresses from multiple categories—Male, Female, and Kids—add them to a virtual cart, and generate a final bill inclusive of Goods and Services Tax (GST).
This system is built using core Java programming concepts and Object-Oriented Programming (OOP) principles, alongside GUI components in Java Swing. Key features include dress selection, cart management, the ability to add new dress entries dynamically, GST-based billing calculations, and tax invoice generation. All data—such as dress details, categories, prices, and transaction records—are managed and stored efficiently, ensuring data persistence, accuracy, and easy retrieval.
A standout feature of the project is its modern and vibrant interface, incorporating emojis, images, and color-themed panels to enhance user experience. The system is scalable for future enhancements like user login, sales reporting, and invoice exports.
